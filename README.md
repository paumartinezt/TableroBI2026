# Business Intelligence Dashboard – Final Project Example

This repository contains the base structure for a Streamlit dashboard developed as part of the **Business Intelligence specialization course** at **Universidad Panamericana, Campus Mixcoac**.

The objective of this repository is to serve as a **reference project and starting point** for building the final project dashboard. It includes the basic structure and example components that can be used to develop an interactive data visualization application using **Python and Streamlit**.

---

## Project Purpose

The purpose of this repository is to provide a working example that demonstrates how a Business Intelligence dashboard can be developed using modern data tools.

This project illustrates:

- How to structure a repository for a BI dashboard
- How to load and process datasets using Python
- How to create interactive visualizations
- How to build a web-based dashboard using Streamlit

The repository is intended to guide the activities required for the **final project of the course**, serving as a practical implementation of concepts covered during the semester.
